JSX & React Native

About me

About react native:

The engineers at facebook designed a new language called "JSX" and a library for jsx called "React native"
to easily work on their facebook mobile apps.

JSX is a combination of html and javaScript method of coding.

Components:

In react native the components always start from uppercase letters.

Component is the base class from which all other components are from.

App:

A function called render() is used in the App component. this function displays the things returned by it.
the render function is the only function in app component which doesn't
have javaScript.

In JSX we can use javaScript inside the flower brackets.

In the some components we can see two flower brackets. one inside another{{}};
the first one is to address that we are using javaScript and
the second one is for using it;

some components have properties such as: style.
